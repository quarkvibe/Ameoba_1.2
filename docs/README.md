## üèóÔ∏è Architecture Overview

Amoeba follows a **cellular architecture** - like a biological cell:

```
MEMBRANE ‚Üí Protection (auth, rate limiting, validation)
NUCLEUS ‚Üí Core intelligence (health guardian, reproduction)
RIBOSOMES ‚Üí Request handling (23 route modules)
GOLGI ‚Üí Processing (27 service modules)
MITOCHONDRIA ‚Üí Data storage (universal, swappable)
CILIA ‚Üí External integrations (OpenAI, Twilio, etc.)
```

**Key Principle:** One service (blob), multiple access points (cilia)

**The Three Drives of Life:**
1. **Self-Preservation** - Health monitoring, auto-healing
2. **Self-Modification** - AI adds features with approval
3. **Self-Reproduction** - Spawns children for efficiency
